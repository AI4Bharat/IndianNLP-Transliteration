# IndianNLP Deep-Transliteration

Open-source transliteration engine for Indian Subcontinent Languages, powered by Deep Learning.

**Supported Languages**

* Bengali - বাংলা
* Gujarati - ગુજરાતી
* Hindi - हिंदी
* Kannada - ಕನ್ನಡ
* Konkani Goan - कोंकणी
* Maithili - मैथिली
* Malayalam - മലയാളം
* Marathi - मराठी
* Panjabi Eastern - ਪੰਜਾਬੀ
* Sindhi - سنڌي‎
* Sinhala - සිංහල
* Tamil - தமிழ்
* Telugu - తెలుగు
* Urdu - اُردُو
